# praksis-IS-302

CV-nettside for Silje Solberg, laget som en del av IS-302.

## Filer

| Fil | Beskrivelse |
|---|---|
| `index.html` | Hoved-CV-siden |
| `style.css` | Stil og layout |

## Innhold på siden

- **Om meg** – kort presentasjon
- **Utdanning** – studier og skole
- **Erfaring** – relevant arbeids- og frivilligerfaring
- **Ferdigheter** – tekniske og faglige ferdigheter
- **Kontakt** – kontaktskjema

## Tilpasning

Rediger `index.html` for å oppdatere personlig informasjon, e-post, LinkedIn-lenke og innhold.
For å aktivere kontaktskjemaet, registrer deg på [Formspree](https://formspree.io) og bytt ut `your-form-id` i skjemaets `action`-attributt.
